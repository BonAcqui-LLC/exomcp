ExoMCP is the MCP and tool-call security layer of the Anti-Illogical product stack — the point where agentic AI meets independent instrumentation.

**The problem:** Model Context Protocol is becoming the standard interface for AI tool connectivity. It is powerful infrastructure. It is also an uninstrumented attack surface.

**The category:** AI behavioral EDR. Independent runtime monitoring for AI systems operating under declared policy, logic, and authority boundaries.
